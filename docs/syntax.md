# Синтаксис языка "Медведь"

Этот документ описывает базовый синтаксис и ключевые слова языка программирования "Медведь".

## 1. Переменные

Для объявления переменных используется ключевое слово `переменная`.

**Синтаксис:**
`переменная <имя_переменной> = <значение>`

**Примеры:**
```
переменная имя = "Большой Медведь"
переменная возраст = 15
переменная активен = истина
```

## 2. Функции

Для объявления функций используется ключевое слово `функция`.

**Синтаксис:**
```
функция <имя_функции>(<аргументы>):
    <тело_функции>
```

**Пример:**
```
функция приветствовать(имя):
    вывод("Привет, " + имя)

приветствовать("Миша")
```

## 3. Условные операторы

Для условных конструкций используются ключевые слова `если`, `иначе если`, и `иначе`.

**Синтаксис:**
```
если <условие>:
    <блок_кода>
иначе если <другое_условие>:
    <другой_блок_кода>
иначе:
    <финальный_блок_кода>
```

**Пример:**
```
переменная возраст = 10
если возраст >= 18:
    вывод("Совершеннолетний медведь")
иначе:
    вывод("Медвежонок")
```

## 4. Циклы

### Цикл `пока` (while)

Для создания циклов используется ключевое слово `пока`.

**Синтаксис:**
```
пока <условие>:
    <тело_цикла>
```

**Пример:**
```
переменная счетчик = 0
пока счетчик < 5:
    вывод(счетчик)
    счетчик = счетчик + 1
```

## 5. Встроенные функции

### Вывод в консоль

Для вывода информации в консоль используется функция `вывод()`.

**Пример:**
```
вывод("Привет, мир!")
вывод(42)
```

## 6. Типы данных

- **Строки:** `"Привет"`
- **Числа:** `10`, `3.14`
- **Булевы:** `истина`, `ложь`

## 7. Операторы

- **Арифметические:** `+`, `-`, `*`, `/`
- **Сравнения:** `==`, `!=`, `<`, `>`, `<=`, `>=`
- **Присваивания:** `=`
