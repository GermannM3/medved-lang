# Установка и запуск проекта

## 1. Клонирование репозитория

Скачайте репозиторий:

```bash
git clone https://github.com/yourusername/medved-lang.git
cd medved-lang


2. Установка зависимостей

На данный момент проект использует Python 3. Установите его с помощью:

pip install -r requirements.txt


3. Запуск проекта

Для запуска используйте команду:

python src/main.py


4. Тестирование

Для тестирования используйте:

pytest
