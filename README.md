# 🐻 Медведь — Язык Программирования на Кириллице

**Медведь** — это полноценный язык программирования, который пишется исключительно кириллицей и на русском языке. Никаких `self`, `def`, `import` — только честный русский код!

## ✨ Особенности

- **Полностью кириллический синтаксис** — все ключевые слова на русском
- **Самостоятельный язык** — собственный лексер, парсер и исполнитель
- **Поддержка основных конструкций**:
  - Переменные и типы данных (числа, строки, булевы значения, списки)
  - Функции с параметрами и возвратом значений
  - Условные операторы (`если`/`иначе`)
  - Циклы (`для`, `пока`)
  - Рекурсия
  - Встроенные функции (`печать`, `ввод`, `длина`, `диапазон`, `число`, `строка`)

## 🚀 Быстрый старт

### Вариант 1: Установка из .deb пакета (рекомендуется)

Скачайте и установите пакет для Debian/Ubuntu:

```bash
# Скачайте пакет из раздела Releases
wget https://github.com/yourusername/medved/releases/download/v1.0/medved_1.0_amd64.deb

# Установите пакет
sudo dpkg -i medved_1.0_amd64.deb

# Запустите программу
медведь /usr/share/медведь/examples/привет.мед

# Или вызовите справку
медведь-помощь
```

**Преимущество:** Не требует установленного Python! Бинарный файл полностью самодостаточен.

### Вариант 2: Запуск из исходников

```bash
git clone https://github.com/yourusername/medved.git
cd medved
python3 src/главный.py examples/привет.мед
```

### Вариант 3: Использование бинарника напрямую

```bash
# Скомпилированный бинарник находится в dist/
./dist/медведь examples/привет.мед
```

## 📝 Пример кода

```медведь
# Привет, Мир!
печать("Привет, Мир!")

# Переменные
имя = "Медведь"
версия = 1.0

# Функции
функция приветствовать(кто):
    печать("Привет, " + кто + "!")
    возврат "ok"

приветствовать("Мир")

# Условия
число = 42
если число > 40:
    печать("Число больше 40")

# Циклы
для i в диапазон(5):
    печать("Считаем: " + строка(i))

# Рекурсия - факториал
функция факториал(n):
    если n <= 1:
        возврат 1
    возврат n * факториал(n - 1)

печать("Факториал 5: " + строка(факториал(5)))
```

## 🔧 Ключевые слова

| Keyword | Описание |
|---------|----------|
| `функция` | Объявление функции |
| `возврат` | Возврат значения из функции |
| `если` / `иначе` | Условный оператор |
| `пока` | Цикл while |
| `для` / `в` | Цикл for |
| `печать` | Вывод на экран |
| `ввод` | Ввод от пользователя |
| `истина` / `ложь` | Булевы значения |
| `ничто` | Null значение |
| `и` / `или` / `не` | Логические операторы |
