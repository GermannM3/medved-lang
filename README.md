# Медведь - Язык программирования

**Автор:** Герман  
**Контакт для связи:** [germannm@vk.com](mailto:germannm@vk.com)

**Медведь** — это новый язык программирования, ориентированный на удобство, производительность и безопасность. Язык сочетает в себе лучшие черты таких языков, как Python, Rust, и C++, а также использует кириллицу для ключевых слов и синтаксиса.

Этот проект предоставляет базовую реализацию языка Медведь, включая компилятор и интерпретатор. В будущем будет добавляться больше функциональности и возможности для разработки полноценных приложений.

## Основные особенности

- **Простота синтаксиса:** Похож на Python, но с использованием русского языка.
- **Производительность:** Язык компилируемый, поддерживает статическую типизацию, управление памятью как в Rust.
- **Совместимость с веб-разработкой:** Возможность создания веб-компонентов и работы с WebAssembly.
- **Поддержка кириллицы:** Весь синтаксис и ключевые слова написаны на русском языке.

## Структура проекта

Проект включает в себя несколько основных компонентов:

- **`src/`** — Исходный код: парсер, компилятор, интерпретатор.
- **`tests/`** — Модульные тесты для проверки работы парсера и компилятора.
- **`docs/`** — Документация и инструкции по использованию.
- **`requirements.txt`** — Зависимости Python (например, для тестирования).
- **`README.md`** — Этот файл с инструкцией по проекту.

Структура кода
1. src/main.py

Это основной файл, который включает в себя точку входа в программу. Здесь происходят компиляция и интерпретация кода на языке Медведь.
2. src/parser.py

Парсер — это компонент, который анализирует код на языке Медведь и преобразует его в промежуточное представление.
3. src/compiler.py

Компилятор преобразует код из промежуточного представления в машинный код, готовый для выполнения.
4. tests/

В этой директории находятся тесты для проверки работоспособности парсера и компилятора.

    test_parser.py — Тесты для парсера.
    test_compiler.py — Тесты для компилятора.

5. docs/

Здесь находятся документация и инструкции по использованию проекта.
